🌾 AgroVision – AI Crop Health & Yield Prediction System
📌 Project Overview

AgroVision is a full-stack AI-powered web application designed to assist farmers in monitoring crop health and predicting agricultural yield using Machine Learning and Deep Learning techniques.

The system enables users to upload crop images for disease detection and enter environmental and soil parameters to estimate crop yield. AgroVision aims to promote data-driven farming practices and improve agricultural productivity.

🎯 Problem Statement

Agriculture is highly dependent on environmental factors such as rainfall, temperature, soil type, and fertilizer usage. Farmers often lack access to intelligent systems that can:

Detect crop diseases at early stages

Predict crop yield accurately

Provide insights based on real-time input data

This leads to reduced productivity, financial losses, and inefficient resource usage.

💡 Proposed Solution

AgroVision provides:

🌿 Crop Disease Detection using Deep Learning (CNN)

📊 Crop Yield Prediction using Machine Learning Regression Models

🧪 Input-based agricultural analysis

🌐 User-friendly web interface for easy access

The system integrates AI models with a web-based frontend and backend to deliver real-time predictions.
